## 一、项目概述
1、项目基本业务概述
- 多种终端访问方式--对应不同的应用场景
- ==运用同一份API接口连接服务器，使用同一份数据库==
	- PC端
	- 移动APP
	- 移动Web（手机浏览器）
	- 微信小程序
	- **PC后台管理**

ps：前四个是给**普通用户**使用、最后一个PC后台管理是提供给**管理员**使用

3、开发模式（前后端分离）
- 前端：基于Vue技术栈的SPA项目
	- 绘制页面
	- ajax**调接口**
- 后端
	- 连接数据库
	- 暴露一些API接口给前端（**写接口**）


4、技术选型
- 前端项目技术栈
	- Vue
	- Vue-router（路由）
	- Element-UI（前端UI组件库）
	- Axios（发送网络数据请求）
	- Echarts（绘制图形报表）

- 后端项目技术栈
	- Node.js
	- Express
	- Jwt（Json Web Token 状态保持工具）
	- ilims
	- Sequelize（数据库框架）

## 二、项目初始化
- 前端项目初始化
	1. 安装Vue脚手架
	2. 通过Vue脚手架创建项目
	3. 配置Vue-router
	4. 配置Element-UI组件库
	5. 配置axios库
	6. 初始化git远程仓库（源代码管理）
	7. 将本地项目托管至Github

- 后端项目初始化
	1. 安装数据库环境
	2. 安装Node.js环境
	3. 配置项目相关信息
	4. 启动项目
	5. 使用Postman测试后台项目接口是否正常
